# Portafolio - Javier Sandoval

Este es mi portafolio desarrollado con **React**, **Vite** y **Three Fiber**. Incluye una escena 3D interactiva con modelos optimizados.

## Tecnologías utilizadas
* React.js
* Three.js (React Three Fiber)
* Vercel (para el despliegue)

## Instalación
1. Clonar el repo: `git clone https://github.com/jasandoval454/Portafolio-Nuevo-Javier-Sandoval.git`
2. Instalar dependencias: `npm install`
3. Correr proyecto: `npm run dev`
